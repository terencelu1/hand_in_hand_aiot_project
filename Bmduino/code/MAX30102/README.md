# MAX30102 心率血氧感測器程式

## 功能說明
- 讀取心率 (Heart Rate)
- 讀取血氧 (SpO2)
- 透過 Serial 輸出數據

## 硬體連接
根據 `BMduino 腳位定義.md`：
- **VCC** => VDD1
- **GND** => GND
- **SDA** => SDA1
- **SCL** => SCL1

## 使用說明

### 1. 上傳程式
1. 打開 Arduino IDE
2. 打開 `MAX30102.ino` 檔案
3. 選擇正確的開發板（BMduino）
4. 選擇正確的 COM 埠
5. 點擊上傳

### 2. 查看數據
1. 打開 Serial Monitor（波特率：115200）
2. 將手指輕輕放在感測器上
3. 等待幾秒鐘讓感測器穩定
4. 觀察 Serial Monitor 輸出的心率與血氧數據

## 輸出格式
```
IR=123456, Red=123456, 心率=72 BPM, 血氧=98%
```

## 注意事項
1. 手指必須穩定放置在感測器上，不要移動
2. 初次使用需要等待幾秒讓數據穩定
3. 如果顯示「未偵測到手指」，請檢查手指是否正確放置
4. 感測器需要良好的接觸才能獲得準確讀數

## 檔案結構
- `MAX30102.ino` - 主程式
- `MAX30105.h` / `MAX30105.cpp` - MAX30102 感測器驅動庫
- `heartRate.h` / `heartRate.cpp` - 心率檢測演算法
- `spo2_algorithm.h` / `spo2_algorithm.cpp` - 血氧計算演算法
